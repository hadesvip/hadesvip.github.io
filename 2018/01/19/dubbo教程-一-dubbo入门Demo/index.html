dubbo教程(一):dubbo入门Demo

2018-01-19 java dubbo, java, rpc, soa 0 Comments

什么是dubbo?

  • 官网: dubbo.io
  • 官方文档－用户手册:　http://dubbo.io/books/dubbo-user-book/
  • 高性能和透明化的RPC远程服务调用方案
  • SOA 服务治理方案

dubbo能做什么?

  • 远程化的远程方法调用d
    • 就像调用本地方法一样调用远程方法（动态代理）
    • 只需要简单的配置，没有任何API侵入
  • 软负载均衡及容错机制
    • 可在内网代替F5等硬件负载均衡器
  • 服务自动注册与发现
    • 不需要写死服务提供方地址，注册中心基于接口名查询服务提供者的IP地址，并且能够平滑添加或者删除服务提供者
